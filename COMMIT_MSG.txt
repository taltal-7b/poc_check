Simplify role management to 4 role types

Role types:
- Manager: Full project and member management access
- Developer: Create/edit/delete tickets, edit wiki, log time
- Reporter: Create tickets, view all, comment only
- Viewer: View all content, comment only

Changes:
- Replace complex permission matrix with simple role type selector
- Add role type descriptions for clarity
- Update role list table to show role type and description
- Auto-assign permissions based on selected role type
