# 画面/テンプレート/アセット

## テンプレート構成

- レイアウト: `app/views/layouts/`
- 主要画面: `app/views/**`
- 画面は部分テンプレート（`_*.erb`）に分割される。

## 代表的な画面領域（例）

- プロジェクト: 一覧/詳細/設定
- 課題: 一覧/詳細/編集/一括編集
- 管理: ロール/権限、トラッカー、ステータス、ワークフロー
- Wiki: 閲覧/編集/履歴/差分

## アセット

- JS: `public/javascripts/`
- CSS: `public/stylesheets/`
- テーマ: `public/themes/`
- ヘルプ（静的HTML）: `public/help/`（日本語含む）
