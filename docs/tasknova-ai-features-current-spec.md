# TaskNova AI機能 現行仕様書

## 1. 機能一覧

| 機能 | 実行契機 | 対象 | 主な出力 |
| --- | --- | --- | --- |
| AI進捗要約/指示 | プロジェクト画面から手動実行 | 未完了チケット | 進捗、注意点、次の指示 |
| AI週次レポート | プロジェクト画面から手動実行 | 直近1週間に作成、更新、期日到来したチケット | 今週の動き、リスク、来週の推奨アクション |
| AIボトルネック検知 | プロジェクト画面から手動実行 | 期日超過中の未完了チケット、期日超過後に完了したチケット | 遅延傾向、要因、改善策 |
| AIタスク指示 | プロジェクト画面から手動実行 | 未完了チケット | 優先順位、担当者別アクション、確認事項 |
| 期限前AI要約通知 | スケジューラーで自動実行 | 期日が近い未完了チケット | 担当者向け要約メール |

現行実装に、期日超過済みチケットを対象にしたAIメール通知はない。遅延チケットの分析は「AIボトルネック検知」が担う。

## 2. プロジェクトAIアクション

各AIアクションはプロジェクト詳細画面から実行する。実行者は対象プロジェクトのメンバーである必要がある。週次レポート、ボトルネック検知、タスク指示は、対象プロジェクトでチケットトラッキングが有効であることも条件となる。

| 機能 | 入力の要点 | 既定取得上限 |
| --- | --- | ---: |
| AI進捗要約/指示 | プロジェクト概要、メンバー、未完了チケット、説明、直近コメント | 100件 |
| AI週次レポート | プロジェクト情報、週次対象チケット、履歴、関連情報、工数、カスタムフィールド | 200件 |
| AIボトルネック検知 | 遅延中チケット、遅延完了チケット、遅延日数、担当者別等の集計、工数、直近コメント | 各30件 |
| AIタスク指示 | 未完了チケット、担当者別等の集計、期日超過、期日間近、更新停滞、未着手フラグ | 80件 |

## 3. 期限前AI要約メール通知

期限前AI要約通知は、期日が近い未完了チケットの説明とコメントをAIで要約し、担当者へメール送信する。

| 項目 | 現行仕様 |
| --- | --- |
| 既定実行時刻 | `Asia/Tokyo` の7時00分 |
| 既定対象期間 | 期日が3日以内に到来 |
| 対象チケット | 未完了、期日あり、担当者または担当グループあり |
| 通知先 | 有効ユーザー、メールアドレスあり、メール通知を無効化していない担当者または担当グループ所属ユーザー |
| メール内容 | プロジェクト名、チケット番号、件名、期日、URL、AI要約 |

要約入力には、チケット件名、プロジェクト名、ステータス、進捗率、期日、説明、公開コメントを含める。

## 4. OpenAI連携とプロンプト

バックエンドからOpenAI Responses APIを呼び出す。既定モデルは `gpt-5-nano`。レスポンスは文章として画面またはメールに表示する。応答が空の場合はエラーとし、出力上限で未完了になった場合は注意文を付与する。

各機能には既定プロンプトがあり、役割と出力観点を指定する。実行時は `settings` テーブルの値を優先し、未設定時は環境変数または設定既定値を使用する。

| 機能 | 既定プロンプト |
| --- | --- |
| 期限前AI要約通知 | あなたはプロジェクト管理ツールのチケット内容を担当者向けに要約するアシスタントです。タイトル、説明、コメントから、期限前に担当者が確認すべき状況・未解決点・次の行動を日本語で簡潔にまとめてください。 |
| AI進捗要約/指示 | あなたはプロジェクト管理ツールの進捗を要約するアシスタントです。プロジェクト概要、メンバー、未完了チケットの内容から、進捗状況、注意点、次の指示を日本語で簡潔にまとめてください。 |
| AI週次レポート | あなたはプロジェクト管理ツールの週次レポートを作成するアシスタントです。プロジェクト情報、直近1週間に作成・更新・期日到来したチケット情報から、今週の動き、完了/進行中の状況、リスク、来週の推奨アクションを日本語で簡潔にまとめてください。 |
| AIボトルネック検知 | あなたはチケット管理システムのボトルネック検知アシスタントです。未完了で期日超過しているチケットと、過去に期日超過後に完了したチケットの情報から、担当者・内容・作業分類・カテゴリ・見積・進捗・履歴に見られる法則性、遅延要因、改善策を日本語で簡潔に示してください。個人攻撃ではなく、プロセス改善につながる表現にしてください。 |
| AIタスク指示 | あなたはチケット管理システムのタスク指示アシスタントです。プロジェクト情報、メンバー、未完了チケットの状況から、優先順位、担当者ごとの次アクション、確認すべき不明点、今日または次に着手すべきタスクを日本語で具体的に示してください。 |

各機能には対象件数、コメント件数、説明文字数、コメント文字数、総入力文字数、出力トークン数の上限がある。詳細な材料はプロンプト本文ではなく、機能ごとの入力データとして渡す。
