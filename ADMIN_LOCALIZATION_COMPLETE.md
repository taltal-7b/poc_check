## 🚀 アクセス方法

### ナビゲーションから
1. 管理者ユーザーでログイン
2. ナビゲーションバーの「管理」アイコン（盾マーク）をクリック
3. 管理ダッシュボードから各機能にアクセス

### 直接URL
```
http://localhost:5173/admin          # 管理ダッシュボード
http://localhost:5173/admin/users    # ユーザー管理
http://localhost:5173/admin/roles    # ロール管理
http://localhost:5173/admin/groups   # グループ管理
http://localhost:5173/admin/custom-fields  # カスタムフィールド
http://localhost:5173/admin/workflows      # ワークフロー
```
