## デザインシステム

### カラーパレット

#### プライマリーカラー（青系）
- Primary 600: `#0284c7` - ボタン、リンク
- Primary 500: `#0ea5e9` - フォーカス時のリング
- Primary 700: `#0369a1` - ホバー時

#### ニュートラルカラー（グレー系）
- Gray 50: `#f9fafb` - 背景
- Gray 200: `#e5e7eb` - ボーダー（薄め）
- Gray 300: `#d1d5db` - ボーダー
- Gray 400: `#9ca3af` - プレースホルダー
- Gray 500: `#6b7280` - 補助テキスト
- Gray 700: `#374151` - 通常テキスト
- Gray 900: `#111827` - 見出しテキスト

#### ステータスカラー
- Blue: 情報
- Green: 成功、完了
- Yellow: 警告、進行中
- Orange: 高優先度
- Red: エラー、緊急

### コンポーネントスタイル

#### ボタン

```css
/* プライマリボタン */
.btn-primary
  - 背景: #0284c7 (青)
  - テキスト: 白
  - ホバー: #0369a1 (濃い青)
  - シャドウ: あり

/* セカンダリボタン */
.btn-secondary
  - 背景: 白
  - テキスト: #374151 (グレー)
  - ボーダー: #d1d5db (グレー)
  - ホバー: #f9fafb (薄いグレー背景)
  - シャドウ: あり
```

#### 入力フィールド

```css
.input
  - 背景: 白
  - ボーダー: #d1d5db (グレー)
  - テキスト: #111827 (黒)
  - プレースホルダー: #9ca3af (薄いグレー)
  - フォーカス時:
    - リング: 2px #0ea5e9 (青)
    - ボーダー: #0ea5e9 (青)
  - シャドウ: 軽いシャドウ
  - トランジション: スムーズ
```

#### カード

```css
.card
  - 背景: 白
  - ボーダー: 1px #e5e7eb (薄いグレー)
  - シャドウ: 軽いシャドウ
  - 角丸: 0.5rem
  - パディング: 1.5rem
```

## 今後の拡張

このデザインシステムは拡張可能です：

### 追加できる要素

```css
/* タブ */
.tab {
  @apply px-4 py-2 border-b-2 border-transparent hover:border-gray-300 transition-colors;
}

.tab-active {
  @apply border-primary-600 text-primary-600;
}

/* バッジ */
.badge {
  @apply px-2 py-1 text-xs font-medium rounded-full;
}

/* アラート */
.alert {
  @apply p-4 rounded-md border;
}

.alert-info {
  @apply bg-blue-50 border-blue-200 text-blue-800;
}

.alert-success {
  @apply bg-green-50 border-green-200 text-green-800;
}
```

## トラブルシューティング

### スタイルが反映されない場合

#### 1. ブラウザキャッシュをクリア

```
F12 → Application → Clear site data
```

#### 2. コンテナを完全再起動

```powershell
cd C:\Users\kings\poc_check
docker-compose restart
```

#### 3. 強制リビルド

```powershell
docker-compose down
docker-compose up -d --build
```

### 一部のコンポーネントだけ古いスタイルの場合

Tailwind CSSのキャッシュをクリアして再ビルド：

```powershell
# フロントエンドコンテナ内で
docker exec pm_frontend npm run build
```
